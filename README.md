# React Hero

## Quick Start

在本地端執行專案：

1. clone GitHub 專案。

```bash
git clone https://github.com/Airwavess/react-hero
```

2. 在專案資料夾中執行 `yarn` 安裝相依套件。

```bash
yarn
```

3. 執行 `yarn dev`，至瀏覽器中輸入 `localhost:3000` 進入網站。

```
yarn start
```
